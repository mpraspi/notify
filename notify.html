Mediapipe Raspbian News - 11.05.2017

   Thank-you for your continued support of Mediapipe Solutions.  This update moves the version up to V6.1.0 as it is a major overhaul of the update system to accommodate the new System Retain feature.

  Mediapipe Raspbian V6.1.0
  Fixes:

  • Raspbian system security updates (Incl. Krack WiFi vulnerability)
  • Various Mediapipe Tools improvements and fixes
  • Re-vamped Update System; bug fixes; more error-checks; improved user interface and logging

  Features:

  • New backup option -> 'System Retain' - keep System preferences and/or user  selected Kodi settings after resetting the system
  • New MP-Tool -> 'Kodi-Clear Cache' - remove Kodi temporary and cache files; can help fix 'stuck' addons
  • New Kodi Add-on -> 'Supremacy' - Movies, TV, 24-7 streaming, and more
